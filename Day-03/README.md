### 🔹 DELETE, DROP & TRUNCATE

Learned the differences between these three SQL commands and when each should be used.

| Command | Purpose |
|----------|---------|
| **DELETE** | Removes selected records from a table while keeping its structure. |
| **TRUNCATE** | Removes all records from a table but keeps the table structure. |
| **DROP** | Permanently removes both the table structure and its data. |
