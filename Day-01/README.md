# 🚀 Day 1 | SQL & Microsoft Excel Fundamentals

Welcome to **Day 1** of my **Meezan | Data Analytics Journey**.

Today marks the beginning of my journey toward becoming a **Data Analyst**. I started by learning the fundamentals of **SQL** and **Microsoft Excel**, two of the most essential tools used by data analysts.

The objective of Day 1 was to build a strong foundation by understanding how data is stored, organized, and managed before moving on to advanced analytics concepts.

## 📊 Microsoft Excel Fundamentals

### Topics Covered

- Introduction to Microsoft Excel
- Creating Workbooks
- Understanding the Excel Interface
- Language Preferences
- Setting English as the Display Language

### 💡 What I Learned

Microsoft Excel is a powerful spreadsheet application available on **Desktop**, **Web**, and **Mobile** platforms. It is widely used to organize, analyze, and visualize data.

Some common use cases include:

- 📊 Data Analysis
- 📈 Business Dashboards
- 💰 Financial Modeling
- 📅 Project Management
- 📉 Statistical Analysis
- 📋 Reporting & Data Visualization

### ✅ Practical Activities

- Opened Microsoft Excel
- Created my first workbook
- Explored the Excel interface
- Set the display language to English
- Learned the purpose of the main tabs and tools
