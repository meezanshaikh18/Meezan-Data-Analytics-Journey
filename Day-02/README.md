# 🚀 Day 2 | SQL DML, DDL & Microsoft Excel Essentials

Welcome to **Day 2** of my **Meezan | Data Analytics Journey**.

Today, I continued strengthening my fundamentals in **SQL** and **Microsoft Excel** by learning more about database operations, practicing SQL queries, and exploring essential spreadsheet concepts.

The focus of Day 2 was to understand how data can be created, modified, deleted, and managed in SQL while also learning the basics of data representation and formatting in Microsoft Excel.

---

# 📚 What I Learned

## 🗄️ SQL

### Topics Covered

### 🔹 DELETE Command

* Learned how to delete records using the `DELETE` command.
* Practiced deleting specific rows using `TOP` with appropriate conditions.
* Created multiple duplicate records and practiced deleting selected rows using `TOP`.

### 🔹 Hands-on SQL Practice

* Created a **Student** table.
* Inserted **12 records** into the table.
* Retrieved data using `SELECT`.
* Updated records using `UPDATE`.
* Deleted records using `DELETE`.
* Practiced writing and executing SQL queries independently.

### 🔹 DML (Data Manipulation Language)

Learned the commands used to manipulate data inside database tables:

* INSERT
* SELECT
* UPDATE
* DELETE

### 🔹 DDL (Data Definition Language)

Learned the commands used to define and manage database structures:

* CREATE
* ALTER
* DROP
* TRUNCATE
